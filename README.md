# Gravitational Wave Polarization Modes in Modified Gravity

This repository contains Mathematica notebooks developed during my M.Sc. research in gravitational physics.

The notebooks implement symbolic calculations and numerical simulations of gravitational-wave polarization modes in modified theories of gravity.

## Features

* Symbolic derivations of the background and first-order perturbation equations for the Ricci scalar and metric perturbations.
* Computation of gravitational-wave polarization modes through the geodesic deviation equation at first order in metric perturbations.
* Simulations and visualizations of gravitational-wave polarization patterns.

The polarization modes studied include:

* Plus mode
* Cross mode
* Breathing mode
* Longitudinal mode
* Mixed modes
* Circular modes

## Repository Structure

* `notebooks`: Mathematica notebooks containing symbolic derivations and simulations.
* `figures`: GIF animations and figures generated from the simulations.
